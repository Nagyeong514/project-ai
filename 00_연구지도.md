## 우리 환경(중요 — 삽질 방지)

- GPU = **RTX 2080 ×2 (각 8GB, Turing sm75)**, 폐쇄망(외부 API 불가) → 전부 로컬.
- **Turing 철칙: fp16 / 4bit(AWQ·GPTQ·bnb)만 됨. bf16·FP8은 하드웨어 미지원 = 죽음.**
- 반복 함정 정리: ④⑤ 각 보고서의 "환경/방법론 기록" 절 참고
  (gemma2 fp16거부, InternVL 동적타일링 토큰폭증, bnb는 TP 미지원=1장으로 등).
