# 채점 루브릭 (6항목, 1~5점)

(이 파일 = LLM-as-judge 및 사람 검수 공용 루브릭. 계획서 3.3 / 3.5.)
항목별 1·3·5점 기준을 사전 정의. reference-free(근거·타당성 기준)로 운영.

- Faithfulness (충실도, w=0.25)
- Accuracy (정확성, w=0.20)
- Usefulness (유용성, w=0.20)
- CodeSwitch (한·영 혼용 이해, w=0.15)
- Fluency (한국어 자연스러움, w=0.10)
- Format (형식 준수, w=0.10)
