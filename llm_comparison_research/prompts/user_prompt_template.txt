위 입력에 근거하여 현장 암묵지 후보를 정확히 {n_candidates}개 생성하라.
- 각 후보는 입력(VLM 결과·Transcript·용어집)에서 근거를 찾을 수 있어야 한다. 근거 없는 생성은 금지한다.
- 한·영 혼용 표현은 분리하거나 번역하지 말고 섞인 그대로의 의미로 이해하라.
- 출력은 아래 구조의 JSON 배열만, 다른 텍스트 없이 출력하라.

각 후보 객체의 필드:
- "암묵지내용": 현장 노하우 (문자열)
- "근거": 입력의 어느 부분에 근거했는지 (문자열)
- "자기신뢰도": 0~1 사이 실수
- "관련작업단계": 관련된 작업 단계 (문자열)
