# 촬영 가이드 — 연구④ 실험용 영상

목표: **발화/침묵이 섞이고, 노하우가 발화에만 몰리지 않은** 영상 만들기.
(이게 있어야 C 조건 켜고 B vs C 가설검증 가능 — 지금 V03는 발화 90%라 불가)

## 촬영 체크리스트 (1~10)

1. **발화:침묵 ≈ 5:5** 로 찍기 — 말할 땐 말하고, 침묵 땐 진짜 조용히 (V03는 9:1이라 실패).
2. **4칸 다 섞기**: 중요+말 / 중요+침묵 / 안중요+말 / 안중요+침묵.
3. **"중요한데 말없이 시범"** 구간을 의도적으로 몇 개 넣기 (발화 밖에도 노하우 있나 검증 — 핵심).
4. **"중요할 때만 말하기" 금지** — 발화=노하우를 미리 심으면 실험이 무효(조작)가 됨.
5. **후시녹음·자막 더빙 금지**, 카메라 현장 마이크로 (영상-소리 시점 일치 필수).
6. **손·작업 부위가 잘 보이게** — 가까이, 밝게 (노하우는 디테일에 있음).
7. **1080p 이상**, 가로(landscape), 삼각대/거치로 흔들림 최소.
8. **작업 한 종류는 한 번에 쭉** 촬영 (끊지 말고).
9. **3~5개** 찍기 (n=1은 통계 불가).
10. 정답지는 **"내가 말한 곳"이 아니라 "실제 노하우가 있는 곳"으로** 매기기 (조용히 시범한 것도 포함).

## 찍은 뒤
- 파일을 `data/raw/V04.mp4`, `V05.mp4` … 로 넣기.
- 오디오 추출: `ffmpeg -i V04.mp4 -ac 1 -ar 16000 V04.wav`
- `conditions.C.enabled=true` 로 켜고 본실험.
