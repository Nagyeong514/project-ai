## 비교 축별 분석

### A → A′: 배치 파라미터 효과

| 지표 | 변화 |
|------|------|
| WER | 23.5% → 17.7% (-24.7%) |
| CER | 15.9% → 9.3% (-41.3%) |
| RTF | 0.127 → 0.074 (-41.6%, 빨라짐) |
| 할루시네이션/시간 | 231.3 → 128.4 (-44.5%) |

`condition_on_previous_text=False` 단 하나의 파라미터 변경이 정확도·속도·할루시네이션 세 축 모두에서 가장 큰 개선을 만들었음.

### A′ → B: VAD 순수 효과

| 지표 | 변화 |
|------|------|
| WER | 17.7% → 18.8% (+1.1%, 악화) |
| CER | 9.3% → 10.1% (+0.8%, 악화) |
| RTF | 0.074 → 0.086 (+15.7%, 느려짐) |
| 할루시네이션/시간 | 128.4 → 172.0 (+34.0%, 악화) |

무음 4.8% 환경에서 VAD는 모든 지표에서 손해. 이유:
- **속도**: 168.9초(2.8분) VAD 오버헤드가 무음 제거 이득(4.8% = 약 3.5분)을 거의 상쇄
- **정확도**: 161개 청크 경계마다 컨텍스트가 끊기며 경계부 오류 발생
- **할루시네이션**: 청크 단위 추론으로 경계부 할루시네이션 증가

### 타임스탬프 드리프트

세 조건 모두 1.2~1.4초 수준으로 유사. 70분 후반부에서도 크게 누적되지 않음. A가 가장 낮은 것은 `condition_on_previous_text=True`로 타임스탬프 연속성이 유지되기 때문.
