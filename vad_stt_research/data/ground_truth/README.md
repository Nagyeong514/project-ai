# Ground Truth 파일 형식

각 오디오 파일(`{file_id}.wav`)에 대응하는 정답 파일을 `{file_id}.json`으로 저장.

## 스키마

```json
{
  "file_id": "F01",
  "text": "정규화된 전체 전사 텍스트",
  "segments": [
    { "start": 0.52, "end": 3.10 },
    { "start": 5.40, "end": 12.80 }
  ]
}
```

- `text`: 공백·문장부호·숫자 표기 정규화 완료된 전체 스크립트 (WER 비교용)
- `segments`: 수동 라벨링 또는 검증된 정렬 결과 기준의 발화 구간 목록 (타임스탬프 비교용)
  - 단위: 초 (float)
  - 정렬 기준: **세그먼트 레벨** (1차 연구), 단어 레벨은 향후 확장
